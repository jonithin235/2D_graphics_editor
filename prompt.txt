2D GRAPHICS EDITOR MINI PROJECT

PROMPT 1:

I have a mini project in C. The problem statement says to create a 2D graphics editor using * and _. I don't really understand what I am supposed to make. Explain the project in simple words.

RESPONSE:

Explained that the project is like a simple version of MS Paint running in the terminal. Suggested using a 2D character array as a canvas and using * characters to represent drawn shapes while _ characters represent empty space.
